# @faasjs/ant-design

UI components based on [FaasJS](https://faasjs.com) and [Ant Design](https://ant.design).

## Usage

In `@faasjs/ant-design`, we use `FaasItemProps` to provide data struct for components.

`FaasItemProps`'s struct is:

```ts
type FaasItemType =
  'string' | 'string[]' |
  'number' | 'number[]' |
  'boolean'

type FaasItemProps = {
  type: FaasItemTypes
  id: string
  title?: string
  children?: FaasItemProps[]
}
```

### Component special props

Each component's props extends from `FaasItemProps` struct.

#### DescriptionItemProps

```ts
type DescriptionItemProps<T = any> = FaasItemProps & {
  content?: (props: {
    value: T
  }) => JSX.Element
}
```

#### FormItemProps

```ts
type FormItemProps<T = any> = AntdFormItemProps<T> & FaasItemProps & {
  input?: (args: {
    value?: T
    onChange?: (value: T) => void
  }) => JSX.Element
}
```

#### TableItemProps

```ts
type TableItemProps<T = any> = AntdTableColumnProps<T> & FaasItemProps
```
